# User Guide

## Status
This guide is intentionally lightweight in Sprint 1 and will expand as the application becomes runnable in later sprints.

## Planned User Workflows

1. Enter or paste text into the web interface.
2. Choose correction mode such as automatic, T5-only, or RAG-enhanced.
3. Review corrected output, detected errors, and guardrail results.
4. Run batch evaluation or benchmark operations when needed.

## Planned Input and Output Experience

- Users will provide English text with possible grammar mistakes.
- The system will return corrected text and optional diagnostic details.
- Advanced users will be able to inspect prompts, retrieved rules, and evaluation metrics.
